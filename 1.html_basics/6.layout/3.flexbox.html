Main Content

This is the main content section of the page using Flexbox layout.

  • display: flex;와 flex-direction: column;을 사용하여 페이지의 레이아웃을 수직으로 쌓습니다.
  • main 안의 content는 display: flex;를 사용하여 좌측에 내비게이션, 우측에 메인 컨텐츠가 배치됩니다.
  • flex: 1;과 flex: 3;을 사용하여 내비게이션은 1/4, 메인 컨텐츠는 3/4 비율로 화면에 배치됩니다.
  • flex-grow: 1;은 main 영역이 중간 영역을 유연하게 차지하도록 만듭니다.
