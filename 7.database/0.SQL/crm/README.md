# CRM 트랙 (고객/매장/상품/주문)

## 이 트랙의 스토리(시나리오)
당신은 작은 커머스/오프라인 매장을 운영합니다.
고객이 매장에 방문해 주문을 하고, 주문에는 여러 상품이 포함됩니다.

이 트랙의 목표는 아래 질문에 SQL로 답하는 것입니다.
- “매출 1위 고객은 누구인가?”
- “가장 많이 방문한 단골 고객은 누구인가?”
- “매장별 매출/객단가는?”
- “인기 상품 TOP-N은?”
- “상위 상품이 전체의 대부분을 차지하나(파레토)?”

## 시작
```bash
sqlite3 crm.db < crm_init.sql
sqlite3 crm.db
```
